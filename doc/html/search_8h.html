Detailed Description

Search Algorithms Interface.

It covers following search algorithms:

See also
binary_search
Unordered Sequential Search symbol table.
Ordered Sequential Search symbol table.
Binary Search tree symbol table.
Author
Gagaron
Note
All search algorithms only support INTEGER type key and STRING type value , (key, value) pair.
